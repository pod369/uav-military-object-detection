# TankProject

## Folder Layout

- `unity_project/`: Junction link to the active Unity project at `C:\Users\popod\Desktop\Test`.
- `dataset_raw/Final_Dataset/`: Raw images and YOLO `.txt` labels generated by Unity.
- `yolo_dataset/`: Train/val/test dataset prepared from Unity data.
- `real_images/`: Real photos used only for visual prediction. No labels required.
- `real_test_dataset/`: Manually labeled real-photo test dataset.
- `scripts/`: Python scripts for preparation, training, prediction, and evaluation.
- `runs/`: YOLO training, prediction, and evaluation outputs.

## Project Workflow

### Step 1. Generate Unity Dataset

Generate synthetic T-55 images and YOLO label `.txt` files in Unity.

Raw output folder:

```text
C:\Users\popod\Desktop\TankProject\dataset_raw\Final_Dataset
```

Prepare the YOLO dataset:

```powershell
python C:\Users\popod\Desktop\TankProject\scripts\prepare_yolo_t55_dataset.py --clean
```

Preview labels:

```powershell
python C:\Users\popod\Desktop\TankProject\scripts\preview_yolo_labels.py --split train --count 30
```

### Step 2. Train YOLO With Unity Dataset

Run this in Spyder:

```text
C:\Users\popod\Desktop\TankProject\scripts\train_t55_yolo_spyder.py
```

Output model:

```text
C:\Users\popod\Desktop\TankProject\runs\<run_name>\weights\best.pt
```

### Step 3. Make Real Photo Test Data

For visual prediction only, put real photos here:

```text
C:\Users\popod\Desktop\TankProject\real_images
```

For quantitative evaluation, put manually labeled real test data here:

```text
C:\Users\popod\Desktop\TankProject\real_test_dataset\images\test
C:\Users\popod\Desktop\TankProject\real_test_dataset\labels\test
```

Each image needs a matching `.txt` label file. For non-T-55/background images, create an empty `.txt` file.

### Step 4. Test The Trained Model

Evaluate on the Unity test split:

```text
C:\Users\popod\Desktop\TankProject\scripts\evaluate_unity_test_spyder.py
```

Predict on real photos without labels:

```text
C:\Users\popod\Desktop\TankProject\scripts\predict_real_images_spyder.py
```

Evaluate on manually labeled real photos:

```text
C:\Users\popod\Desktop\TankProject\scripts\evaluate_real_test_spyder.py
```
